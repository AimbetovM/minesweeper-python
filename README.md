# minesweeper-python

Минимальная реализация игры «Сапёр» на Python с использованием `tkinter`.

## Запуск

```bash
python3 main.py
```

## Что реализовано

- окно с игровым полем;
- случайная генерация мин;
- подсчёт мин вокруг каждой клетки;
- открытие клетки по клику;
- завершение игры при нажатии на мину (и показ всех мин).

## Структура проекта

- `main.py` — весь код игры:
  - создание интерфейса;
  - генерация и хранение поля;
  - логика открытия клеток и проверки победы/поражения.
- `README.md` — краткое описание и инструкция по запуску.
